🗺️ 用户旅程情绪地图

从开箱到日常使用，用户体验情绪呈明显下滑趋势。开箱阶段情绪最高（7.5/10），日常使用阶段情绪最低（4.5/10），过曝和闪退是情绪低谷。

📦 阶段一：开箱体验
📦
开箱体验（66 条数据）
7.5/10

平均总分 75/80，68% 用户"符合期望"，18% "超出期望"。整体开箱体验良好，4G 功能和小巧外观是核心惊喜点。

✅ 爽点 Top 3

4G 免流功能惊艳（21 条提及）
用户最常提到的亮点，认为 4G 远程查看是差异化卖点
"4G免流功能很赞" / "4G模块直接在主机里面更方便实用"
小巧轻便外观精致（16 条提及）
尺寸和手感超出预期，不遮挡视野
"外观小巧轻便不容易遮挡视野" / "非常精致质感好"
TF 卡即插即用（90%+ 一次识别）
卡槽位置明显，插拔顺畅，一次识别成功率高
"一眼就能看到" / "插拔顺畅" / "一次识别成功"

❌ 槽点 Top 3

外观瑕疵（8 条）
摄像头内有颗粒灰尘、卡扣印迹、维修标等
"做工有问题，摄像头里面有很大的颗粒灰尘"

"维修标影响美观和信任感"

"纸质版操作指南无产品参数介绍"

🎯 优先级改进建议
P0 紧急
画面过曝问题（65 条，9.7%）—— 影响核心体验 贯穿点火抓拍、实时查看、录像回放、缩时录影全场景。重点排查哨兵/缩时模式熄火后 AE 逻辑，ISP 曝光控制作为下个版本核心攻坚方向。
P0 紧急
APP 闪退 + 下载链路（28 条）—— 影响用户留存 下载视频场景闪退率极高，9/11 条闪退与下载相关。排查下载模块内存管理，重构下载流程（状态管理+文件路径+完成通知+后台下载）。
P0 紧急
"其他用户正在观看直播"误提示（4 条）—— 信任风险 虽为误提示，但用户感知为隐私泄露风险。立即排查开流锁机制和并发判断逻辑。
P1 重要
哨兵灵敏度两极分化（54 条）—— 影响核心卖点 误报 31 条（树动/下雨/灯光）+ 漏报 23 条（中档太弱）。重新校准三档灵敏度阈值，优化人形/车辆检测算法。
P1 重要
底座角度调节受限（19 条）—— 影响安装体验 上倾角调节范围不足，建议增加左右旋转功能或分体式底座设计。
P1 重要
GPS 轨迹不完整（19 条）—— 影响功能信任 短行程丢失、起点偏差大。优化首次定位速度和信号弱场景的轨迹补全策略。
P2 优化
视频回放交互优化（进度条异常 3 条 + 卡顿 15 条） 排查 TS 片段索引和时间戳映射，优化解码和缓存策略。
P2 优化
说明书完善 + 降压线规范化 补充产品参数、配网指引；统一接线颜色和保险丝规格标识。
💡 关键洞察

🔴 用户最关心的 3 个核心问题

  • 画面过曝—— 贯穿全场景的第一大问题，65 条反馈占比 9.7%，直接影响"能不能看清"
  • 哨兵模式可靠性—— 误报漏报并存，54 条反馈占比 8.0%，影响"能不能信任"
  • 视频下载体验—— 下载慢+闪退+找不到视频，28 条反馈，影响"能不能用"

⚠️ 最可能影响口碑的 3 个风险点

  • "其他用户正在观看直播"误提示—— 用户感知为隐私泄露，社交媒体传播风险极高
  • 设备发热+存储卡烧毁—— 26°C 环境连续测试 1.5 小时即发烫休眠，存储卡烧坏
  • 过曝"重启才能恢复"—— 用户需远程重启设备才能恢复正常画面，使用门槛高

✨ 最可能带来推荐的 3 个亮点

  • 4G 远程查看—— 第一亮点（21 条提及），"随时随地看车"是核心差异化卖点
  • 小巧精致外观—— 16 条提及，"不遮挡视野"+"质感好"形成良好第一印象
  • 哨兵+缩时+云录三位一体—— 停车安全闭环理念获用户认可，期待功能完善
📉 体验情绪曲线总结

用户从开箱的期待与惊喜（7.5分） → 安装的中性偏正（6.5分） → 日常使用的明显不满（4.5分），情绪下滑 40%。
核心结论：硬件设计和开箱体验达标，但软件稳定性和图像质量是最大短板，亟需在下一版本重点突破 ISP 调优和 APP 稳定性。
